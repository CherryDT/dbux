# @dbux/babel-plugin

`@dbux/babel-plugin` instruments JavaScript programs and injects the [`@dbux/runtime`](../dbux-runtime) to record everything they do.

* Install the following packages: `@dbux/babel-plugin` `@dbux/runtime` `core-js@3`
