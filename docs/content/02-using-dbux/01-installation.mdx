# Installation

import Img from '@src/components/Img';


:::note
Currently, Dbux only provides a UI through the Dbux VSCode extension.
:::

1. Install the [Dbux VSCode Extension](https://marketplace.visualstudio.com/items?itemName=Domi.dbux-code)
3. Select the Dbux Extension -> Click the "Start Dbux" button
   * <Img src="screens/start_dbux.png" zoomable={true} width="300px" />
4. If necessary (e.g. after installation), wait for dependencies to update.
   * <Img src="screens/installing_dependencies.png" zoomable={true} width="300px" />
5. The Dbux VSCode Extension is now ready [to use](../using-dbux/overview)!
   * <Img src="screens/dbux-ready.png" zoomable={true} width="150px" />
